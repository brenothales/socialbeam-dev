## Socialbeam
1. Social Network on Ruby on Rails
2. Social Discussion Channel Network


## Features implemented
1. Users(Beamers)
2. Private Messaging
3. Wall Stream(Public Stream)
4. News Feeds(Public Notifications)
5. Comments
6. Friendship
